Background vs Sensor Demo

Sensor Transmitter Pattern Comparison Demo
Uses a camera/simulated sensor stream and compares each incoming sample against a rolling background model. Every sample tick updates rolling mean, stddev, and z-score = (sample - mean) / stddev.
